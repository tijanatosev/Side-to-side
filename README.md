## RG110-side-to-side
Side to side

[![Watch the video](pictures/7.png)](https://www.youtube.com/watch?v=R-7GHiPx8OI)

## :package: Instalation
Requirements: glut

1. clone this repository to your machine:
```
git clone https://github.com/MATF-RG17/RG110-side-to-side.git ~/
```
2. compile:
```
cd RG110-side-to-side
make
```
3. run:
```
./main
```
